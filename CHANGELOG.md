## [1.0.0] - 2025-01-07

### Added
- Initial release
- SendInboxMail mail transport driver for Laravel
- Support for Laravel 10.x, 11.x, and 12.x
- Configuration file with environment variable support
- Inline attachment support
- Configurable timeout and retry settings
- Comprehensive test suite
- GitHub Actions CI/CD pipeline
